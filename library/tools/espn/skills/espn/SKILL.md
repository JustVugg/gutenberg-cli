# espn

Generated by Gutenberg. Use this skill to call the espn API through the local `espn` CLI.

## When to use

- The user asks for data exposed by espn (apis).
- An agent task needs structured JSON from https://site.api.espn.com.
- You need to perform a write/destructive operation on espn — always preview first, run with `--yes` only on explicit user confirmation.

## How to invoke

The skill assumes the generated CLI is on PATH (or invoke via `scripts/use-go.sh run ./cmd/espn` from the project directory).

```bash
espn operations
espn call getBasketballNbaScoreboard --json
espn sync --json
espn search "<keyword>" --json
```

## Output contract

- All commands accept `--json` and emit machine-readable output.
- `call` returns `{ dryRun, operation, request, response }`. `dryRun: true` means the call was not executed (write op without `--yes`).
- `search` returns cached records. `sync` populates the local SQLite cache.
