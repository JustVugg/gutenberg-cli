# wikipedia-it

Generated by Gutenberg. Use this skill to call the wikipedia-it API through the local `wikipedia-it` CLI.

## When to use

- The user asks for data exposed by wikipedia-it (api).
- An agent task needs structured JSON from https://it.wikipedia.org.
- You need to perform a write/destructive operation on wikipedia-it — always preview first, run with `--yes` only on explicit user confirmation.

## How to invoke

The skill assumes the generated CLI is on PATH (or invoke via `scripts/use-go.sh run ./cmd/wikipedia-it` from the project directory).

```bash
wikipedia-it operations
wikipedia-it call getPageSummaryBologna --json
wikipedia-it sync --json
wikipedia-it search "<keyword>" --json
```

## Output contract

- All commands accept `--json` and emit machine-readable output.
- `call` returns `{ dryRun, operation, request, response }`. `dryRun: true` means the call was not executed (write op without `--yes`).
- `search` returns cached records. `sync` populates the local SQLite cache.
