# dog-api

Generated by Gutenberg. Use this skill to call the dog-api API through the local `dog-api` CLI.

## When to use

- The user asks for data exposed by dog-api (api).
- An agent task needs structured JSON from https://dog.ceo.
- You need to perform a write/destructive operation on dog-api — always preview first, run with `--yes` only on explicit user confirmation.

## How to invoke

The skill assumes the generated CLI is on PATH (or invoke via `scripts/use-go.sh run ./cmd/dog-api` from the project directory).

```bash
dog-api operations
dog-api call getBreedsListAll --json
dog-api sync --json
dog-api search "<keyword>" --json
```

No authentication required.

## Output contract

- All commands accept `--json` and emit machine-readable output.
- `call` returns `{ dryRun, operation, request, response }`. `dryRun: true` means the call was not executed (write op without `--yes`).
- `search` returns cached records. `sync` populates the local SQLite cache.
