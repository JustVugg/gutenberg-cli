# pokeapi

Generated by Gutenberg. Use this skill to call the pokeapi API through the local `pokeapi` CLI.

## When to use

- The user asks for data exposed by pokeapi (api).
- An agent task needs structured JSON from https://pokeapi.co.
- You need to perform a write/destructive operation on pokeapi — always preview first, run with `--yes` only on explicit user confirmation.

## How to invoke

The skill assumes the generated CLI is on PATH (or invoke via `scripts/use-go.sh run ./cmd/pokeapi` from the project directory).

```bash
pokeapi operations
pokeapi call getPokemon --json
pokeapi sync --json
pokeapi search "<keyword>" --json
```

## Output contract

- All commands accept `--json` and emit machine-readable output.
- `call` returns `{ dryRun, operation, request, response }`. `dryRun: true` means the call was not executed (write op without `--yes`).
- `search` returns cached records. `sync` populates the local SQLite cache.
