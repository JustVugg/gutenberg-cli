- The user asks for data exposed by tvmaze (people, schedule, search, shows).

- You need to perform a write/destructive operation on tvmaze — always preview first, run with `--yes` only on explicit user confirmation.

## How to invoke

The skill assumes the generated CLI is on PATH (or invoke via `scripts/use-go.sh run ./cmd/tvmaze` from the project directory).

```bash
tvmaze operations
tvmaze call getSearchShows --json
tvmaze sync --json
tvmaze search "<keyword>" --json
```

## Output contract

- All commands accept `--json` and emit machine-readable output.
- `call` returns `{ dryRun, operation, request, response }`. `dryRun: true` means the call was not executed (write op without `--yes`).
- `search` returns cached records. `sync` populates the local SQLite cache.
