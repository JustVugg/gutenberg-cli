# stripe

Generated by Gutenberg. Use this skill to call the stripe API through the local `stripe` CLI.

## When to use

- The user asks for data exposed by stripe (default).
- An agent task needs structured JSON from https://api.stripe.com/.
- You need to perform a write/destructive operation on stripe — always preview first, run with `--yes` only on explicit user confirmation.

## How to invoke

The skill assumes the generated CLI is on PATH (or invoke via `scripts/use-go.sh run ./cmd/stripe` from the project directory).

```bash
stripe operations
stripe call GetAccount --json
stripe sync --json
stripe search "<keyword>" --json
```

Preview a write operation (default behavior, no side effects):

```bash
stripe call PostAccountLinks --data '{"example":true}' --json
```

Execute it (only after explicit user confirmation):

```bash
stripe call PostAccountLinks --data '{"example":true}' --yes --json
```

## Authentication

Set STRIPE_API_KEY in the environment, or pass `--api-key <key>`.

## Output contract

- All commands accept `--json` and emit machine-readable output.
- `call` returns `{ dryRun, operation, request, response }`. `dryRun: true` means the call was not executed (write op without `--yes`).
- `search` returns cached records. `sync` populates the local SQLite cache.
