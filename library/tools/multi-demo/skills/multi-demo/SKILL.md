# Multi-source demo aggregator

Generated by Gutenberg. Use this skill to run the local `multi-demo` aggregator CLI across multiple API sources and return one merged JSON result.

## When to use

- The user asks for a cross-source view covered by espn, open-meteo.
- An agent task needs one command that fans out to multiple APIs and ranks or merges the results.

## How to invoke

```bash
multi-demo sources
multi-demo fan-out --json
multi-demo fan-out --param key=value --json
```

## Sources

- `espn` — espn; operation: getApisSiteV2SportsBasketballNbaScoreboard; base: https://site.api.espn.com
- `open-meteo` — open-meteo; operation: getV1Forecast; base: https://api.open-meteo.com

## Output contract

- `sources` lists the embedded source manifests.
- `fan-out --json` returns merged/ranked items from each source.
- Source-specific credentials use each source env prefix, not the aggregator prefix.
