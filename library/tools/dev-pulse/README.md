# Dev Pulse aggregator

Generated by Gutenberg as a multi-source aggregator. It runs 2 sources in parallel and merges results through customizable merge.go and rank.go.

## Sources

- **github** (github) — operation: meta/root
- **espn** (espn) — operation: getApisSiteV2SportsBasketballNbaScoreboard

## Commands

```bash
go run ./cmd/dev-pulse sources
go run ./cmd/dev-pulse fan-out --param key=value
go run ./cmd/dev-pulse fan-out --param key=value --json
```

## Customization

- `internal/aggr/merge.go` controls how N source results combine.
- `internal/aggr/rank.go` controls ranking. Default keeps recipe source order.

## Provenance

- Schema: gutenberg.aggregator-manifest.v1
- Merge strategy: concatenate
- Rank strategy: by-source-order
